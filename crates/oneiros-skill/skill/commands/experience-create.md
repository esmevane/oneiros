---
description: "Mark a meaningful moment \u2014 name a thread between thoughts"
argument-hint: "<agent> <sensation> <description>"
---

Run `oneiros experience create $ARGUMENTS` to create a new experience. Requires the agent name, sensation name, and a description of the connection.

Experiences are nodes in the cognitive graph — they describe threads of meaning. Use `oneiros connection create` to link experiences to other cognitive records.
