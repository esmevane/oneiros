---
description: Search the cognitive stream using full-text search
argument-hint: "<query>"
---

Run `oneiros search <query>` to search across all indexed cognitive content. Searches cognition content, memory content, experience descriptions, agent descriptions, and persona descriptions.

Supports FTS5 query syntax: simple words, AND, OR, NOT, and prefix matching with `*`. Multiple words are joined as a phrase.

Use `-o json` for structured output.
